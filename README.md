## OVERVIEW
This project is a web application that integrates with the [REST Countries API](https://restcountries.com/) to pull real-time data about countries, including their flags, population, region, capital, and languages. It serves as a modern interface for users to explore global data.

## Key Features
- **View All Countries:** Fetches and displays a list of all countries on load.
- **Search Functionality:** Allows users to search for specific countries by name.
- **Filtering:** Users can filter countries by region (e.g., Africa, Americas, Asia, Europe, Oceania).
- **Detailed View:** Click on a country to see detailed metrics (Border countries, Native name, Currencies, etc.).
- **Responsive Design:** Optimized for Mobile, Tablet, and Desktop.
